📊 Project Overview
This project analyzes Blinkit sales data using Power BI to generate insights on sales performance, outlet types, and product categories.

🔧 Tools Used
- Power BI
- Power Query
- DAX

📈 Key Features
- Interactive dashboard with filters (Outlet Size, Location, Item Type)
- KPIs: Total Sales, Average Sales, Number of Items, Average Rating
- Visualizations: Bar charts, Donut charts, Line charts
